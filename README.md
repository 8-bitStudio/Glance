# Glance
Glance is a Rapid Serial Visual Presentation speed tool built in python to help you read faster!

The average reading speed of an adult is 250 WPM. Glance helps you read faster and trains you to remember more



# Features
Dynamically change your reading speed from 100 words per minute all the way up to 1,000!

Import text files for an easier way to read

A progress bar to indicate how much longer you have left

Auto centers a word then highlights the middle to red to help your eyes stay in the middle

Use arrow keys to skip or go back 10 words

Custom fonts


# How to install
Glance is a tool built purely in python, make sure you have the latest version installed.


If you are on Mac, use the command 'brew install python-tk'


If you are using Linux use: 'sudo apt-get install python3-tk'


if you get errors such as ModuleNotFoundError: No module named 'tkinter' use the command pip install tkinter (Windows)
